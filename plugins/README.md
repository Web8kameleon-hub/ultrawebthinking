# Plugins Extension System

Ky folder është i dedikuar për plugin-e që zgjeron funksionalitetin e sistemit pa prekur trungun.

## Si funksionon
- Çdo plugin është një modul i pavarur.
- Mund të importohen ose ekzekutohen nga skripta jashtë trungut.
- Për të shtuar një plugin të ri, krijo një skedar të ri në këtë folder.

## Shembull përdorimi
Shiko `example-plugin.ts` për një shembull bazë.
